# Demo
Repository for browsing all implementations for the kozen-labs modules.

## Overview
This repository contains links to all example implementations organized in two ways:
- **By Kozen-Labs Module** - Examples grouped by the specific module they demonstrate
- **By Industry** - Examples grouped by industry vertical or use case

All example repositories follow the naming convention with the prefix `demo-` for easy manual searching.
